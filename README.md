Collection of different agent skills that can help with text-to-image and image-to-image workflows.

## Skills

### 1. ideogram-prompt-generator
The first skill in this repository is `ideogram-prompt-generator`, a specialized utility that converts user image ideas into valid, schema-compliant, minified JSON prompts for the Ideogram 4.0 text-to-image model.

### 2. jibrishjaali-philosophical-symbolic-art-prompt-generator
The second skill in this repository is `jibrishjaali-philosophical-symbolic-art-prompt-generator`, which generates highly detailed philosophical and symbolic metaphor art prompts in the style of jibrishjaali on DeviantArt. It is designed to take abstract philosophical ideas and turn them into rich, meaningful image generation prompts utilizing motifs like surreal scales, towering architecture, and classical human forms.
