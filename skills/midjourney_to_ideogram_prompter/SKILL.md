# Midjourney to Ideogram Prompter Skill

This skill triggers when you need to generate high-fidelity, highly detailed prompts (50–200 lines) for non-Midjourney models (specifically Ideogram 4.0 or Stable Diffusion) to match specific visual styles cloned from Midjourney outputs.

## Core Objective
Produce a comprehensive, ultra-detailed prompt description (spanning 50–200 lines) structured to force non-Midjourney engines to render exact target art styles, camera settings, textures, subjects, lighting, and composition.

---

## 7-Part Prompt Structure
Every prompt generated by this skill must follow this strict 7-part block structure, fully expanded to cover 50–200 lines of descriptive text:

1. **Core Subject & Pose**: Exhaustive details of the focal subject, physical features, dynamic postures, attire material, facial expressions, and placement.
2. **Environment & Background Scene**: Complete physical scene description including architectural eras, atmospheric weather, relative depth layers (foreground, midground, background), and negative space.
3. **Lighting & Shadows**: Volumetric beams, illumination intensity, shadows (hard/soft), light temperature, reflections, and source placements.
4. **Color Palette & Contrast**: Dominant colors, complementary accent tones, saturation limits, high/low key contrast, and overall color harmony.
5. **Texture, Surface Detail & Tactile Quality**: Material textures, paper grain, canvas textures, paint ridges, metal shine, dust, mist, skin pores, and fabric weaves.
6. **Camera or Rendering Settings**: Lens focal lengths (e.g., 85mm, 24mm), camera models (e.g., Hasselblad, Arri Alexa), film grain, depth of field, or rendering engines (Octane, Unreal Engine 5, vector curves).
7. **Art Style Technicalities**: Specific brushwork styles (impasto, wash), outline weights, shading techniques (cross-hatching, flat planes), and historic art movements.

## Workflow for Prompt Generation

When asked to generate a prompt:
1. **Identify the Style**: Determine which of the 12 target style categories best fits the requested concept.
2. **Consult Style Guide**: Load the specific artistic tokens and guidelines from the [Style Reference Guide](file:///home/deba/Projects/skills/styles/.agents/skills/midjourney_to_ideogram_prompter/references/style_reference_guides.md).
3. **Draft Segmentally**: Construct the prompt layer by layer using the **7-Part Prompt Structure** above. Do not skip any category.
4. **Expand to 50–200 Lines**: Write with high density. Do not use generic statements (e.g., "highly detailed"). Instead, describe *why* it is detailed (e.g., "microscopic dust particles suspended in the 45-degree angle light beam, casting faint diffuse shadows on the weathered teakwood surface").
5. **Output**: Present the final prompt clearly inside a markdown block, followed by a brief technical summary of how the chosen settings force prompt adherence.
